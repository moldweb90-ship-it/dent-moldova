1) Лейаут

Двухколоночный layout: слева фиксированная колонка фильтров, справа список карточек.

Сайдбар липкий при прокрутке.

DoD:

При скролле фильтры остаются на месте.

Адаптив: ≥1280px два столбца карточек, 1024–1279px - 2, 768–1023px - 2, <768px - 1 и фильтры сворачиваются в off-canvas.

2) Фильтры слева (вертикально)

Блоки: Город, Категории услуг, Ценовой уровень, Статус приема, Страховки/документы, D-Score диапазон, Рейтинг пациента, Доступность по времени.

Кнопки: Применить, Сбросить.

Реактивная фильтрация по onChange с дебаунсом 300 мс.

DoD:

Изменение любого фильтра обновляет список без перезагрузки.

Кнопка Сбросить возвращает дефолт.

3) Поиск

Единая строка поиска над списком: ищет по названию клиники и по услугам.

Дебаунс 300 мс, регистронезависимо, поддержка множественных слов.

DoD:

Поиск "implant" находит клиники с услугой имплантация даже если нет в названии.

4) Карточки клиник

Фото на всю карточку, затемнение overlay 40–60%.

Название клиники и город по центру.

При наведении появляется слой: цена от X, D-Score, мини-шкалы метрик, кнопки Записаться и Цены под карточкой.

Языки убрать из карточки - оставить в верхнем меню.

DoD:

Ховер накладывает плавный overlay 200–300 мс.

Кнопки кликабельны, курсор pointer, ripple/hover-эффект.

Компактные горизонтальные прогресс-бары: Цена, Доверие, Доступность, Обратная связь.

Анимация наполнения при наведении карточки.

DoD:

При hover заполнение шкал анимируется 300–600 мс.

Без дёрганий при массовом рендере.

DoD:

Tooltip закрывается по mouseout и Esc.

По клику на Цены открывается модальное окно: категории услуг слева, прайс-лист справа.

Временно заполнить демо-данными: Имплантация, Костная пластика, Синус-лифтинг, Диагностика, Анестезия, Постоперационное ведение.

Сверху селект валюты MDL/EUR, таблица с ценами: Услуга - От - До - Примечание.

Кнопка Записаться внизу модалки.

DoD:

Esc и клик по фону закрывают модалку.

Фильтрация по категории внутри модалки работает.

Модальное окно с формой: Имя, Телефон, Email, Предпочтительный способ связи, Услуга, Предпочтительная дата/время, Комментарий, Согласие на обработку данных.

Маска телефона MD.

Анимация появления с framer-motion.

DoD:

Валидация обязательных полей.
